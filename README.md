# Example Express

- Production: https://example-express.azobu.app
- Heroku: https://azobu-example-express.herokuapp.com

## API Documentation

| HTTP  | Endpoint | Description              |
| ----- | -------- | ------------------------ |
| `GET` | `/`      | Show welcome message     |
| `GET` | `/hello` | Show hello world message |

## Usage

### Installation

```sh
yarn
```

### Development

```sh
yarn dev
```

### Production

```sh
yarn start
```
